## How to Start (One Simple Way)

### Step 1: Create your project from template

1. Open: **https://github.com/shubham-bhatt50/whatfix-boilerplate**
2. Click green **"Use this template"** ‚Üí **"Create a new repository"**
3. Name your project (e.g., `mirror-admin-panel`)
4. Click **"Create repository"**

### Step 2: Open in Cursor

1. Open **Cursor IDE**
2. Press `Cmd + Shift + P` (Mac) or `Ctrl + Shift + P` (Windows)
3. Type: **"Git: Clone"** and press Enter
4. Paste your new repo URL: `https://github.com/YOUR_USERNAME/your-project-name`
5. Choose where to save it

### Step 3: Install & run

Open Cursor's terminal (`Cmd + J` on Mac, `Ctrl + J` on Windows) and type:

```
npm install
npm run dev
```

### Step 4: View your project

Open **http://localhost:5173** in Chrome. You're done! üéâ
